================================================
 Vision Inspection System - 설치 및 실행 가이드
================================================

【 최초 설치 】
1. Miniconda 설치 (없는 경우)
   https://docs.conda.io/en/latest/miniconda.html

2. install.bat 더블클릭 → 자동 설치 진행
   (인터넷 연결 필요, 약 5~10분 소요)

【 실행 방법 】
- run.bat 더블클릭

【 모델 업데이트 】
1. 새 model.ckpt 파일을 받아서
   models\patchcore_screw\Patchcore\MVTecAD\screw\v0\weights\lightning\
   폴더에 덮어씌우기

2. run.bat 재실행

【 문제 발생 시 】
- 오류 메시지를 캡처하여 담당자에게 문의
- 재설치: install.bat 다시 실행
